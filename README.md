# whitepaper-react
> Automaticly generated library to work with [whitepaper](http://whitepaper.tools/) in React

## Installation
Using yarn:
```js
yarn add --dev --tilde @nitive/whitepaper-react
```

Using npm:
```js
npm install --save-dev --save-prefix=~ @nitive/whitepaper-react
```
